# ``RFIDManager``

A framework for RFID tag reading, writing, and management on Apple platforms.


### Overview

> English ｜ [中文文档](README-zh.md) 

RFIDManager is an RFID SDK designed specifically for Apple platforms, providing comprehensive RFID tag reading, writing, inventory, and locating capabilities. The SDK is developed with modern Swift language, offering a clean and easy-to-use API.


### Platform Support

|Connection\Platform|iOS|iPadOS|macOS|
|:-:|:-:|:-:|:-:|
|Bluetooth|✅|✅|✅|
|USB|❌|❌|✅|

### Core Features

- **Tag Inventory**: Rapidly scan and identify large numbers of RFID tags
- **Tag Read/Write**: Read and write tag memory banks including EPC, TID, User, etc
- **Tag Locating**: RSSI-based tag locating and finding functionality
- **Barcode Scanning**: Integrated barcode scanning capabilities
- **Device Management**: Complete device connection, configuration, and state management
